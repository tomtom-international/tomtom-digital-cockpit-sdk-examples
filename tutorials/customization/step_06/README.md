# Step 6 - Create a custom Theme Component Provider Service

Have a look at the [Theming & Customization documentation](https://developer.tomtom.com/tomtom-digital-cockpit/developers/development/theming-and-customization).

Implement a custom [ThemeComponentProviderService](https://developer.tomtom.com/assets/downloads/tomtom-digital-cockpit/platform-api/1.0.5256/platform_theming_api_service_themecomponentprovider/com.tomtom.ivi.platform.theming.api.service.themecomponentprovider/-theme-component-provider-service/index.html)
to overwrite theming attributes like colors.
