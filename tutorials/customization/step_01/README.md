# Step 1 - Rename the template app

Rename the source files and classes to:
- `CustomActivity` in `CustomActivity.kt`
- `CustomApplication` in `CustomApplication.kt`

Change the package name in the source files to `com.example.ivi.custom.app`.
Change the application name in `res/values/strings.xml`.

Update the package name in `build.gradle.kts`.
Update all mentioned variables in `AndroidManifest.xml`
